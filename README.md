# FitLife - Treningsenter Website

Moderne og responsiv website for FitLife treningssentrum.

## Funksjoner

- **Responsive Design** - Fungerer perfekt på desktop, tablet og mobil
- **Klasseoversikt** - 6 forskjellige treningsklasser med tider
- **Medlemskapspriser** - 3 fleksible medlemskapsalternativer
- **Kontaktinformasjon** - Adresse, telefon og e-post
- **Modern UI** - Vakkert gradient-design med animasjoner

## Klasseoversikt

- 💪 Styrketrening
- 🧘 Yoga
- 🏃 Kardio & HIIT
- 🤸 Pilates
- 🥊 Boksing
- 👥 Gruppekjør

## Medlemskapspriser

1. **Måned**: 299 kr - 2 klasser i uken
2. **Kvartål**: 799 kr - Ubegrenset klasser + personal trener konsultasjon
3. **År**: 2499 kr - Alt inkludert + VIP support

## Teknologi

- HTML5
- CSS3 (med Grid og Flexbox)
- Responsive design (mobile-first)
- Smooth animations og transitions

## Responsive Breakpoints

- Desktop: Alle funksjoner synlig
- Tablet: Optimalisert layout
- Mobil: Kompakt design med full funksjonalitet

## Bruk

Åpne `index.html` i en nettleser for å se nettsiden.

## Personalisering

Du kan enkelt endre:
- Farger 
- Treningsklasser (innhold i classes-seksjonen)
- Priser og medlemskapsvilkår
- Kontaktinformasjon
- Åpningstider
